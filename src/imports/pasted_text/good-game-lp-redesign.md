既存の採用LPを、構成の正しさは残しつつ、AI生成っぽい汎用感を消して、GOOD GAMEらしい実在感のある採用LPへ再設計してください。

## 対象ブランド
- GOOD GAME
- トレーディングカードゲームショップ / EC / 買取 / イベント運営
- 強みは「カードが好き」という熱量だけでなく、店舗運営・査定・在庫管理・イベント・発送などの現場があること
- 目指したいのは、きれいなだけの採用LPではなく、「この店で働くイメージ」が湧く採用LP

## 現状認識
- 情報構造は正しい
- UIの完成度は高い
- ただし全体が“きれいな汎用採用LP”に見え、SaaS的で、GOOD GAME固有の空気が弱い
- いま必要なのは見た目の派手さではなく、店の現場感、実在感、信頼感、業界固有の文法

  4. 募集職種
  5. インタビュー

  10. 最終CTA
- Semantic tokens / spacing / reduced motion / focus-visible など既存の設計基盤は活かす
- アクセシビリティと可読性を損なわない
- ただおしゃれにするのではなく、ブランド固有性を上げる
- 数値・実績・声は捏造しない。不明なものはプレースホルダーにする

## デザインの方向性
- 「eスポーツ風ネオン」「スタートアップSaaS風」「AI生成の丸カード量産」にはしない
- 方向性は「専門店の信頼感 × カード文化の熱量 × 現場の手触り」
- トレカの世界観は、派手な装飾よりも、値札・在庫ラベル・スリーブ・仕分け・棚・イベント卓・発送資材・査定メモのような実務ディテールで表現する
- キービジュアルはイラストより実写優先
- 色は現行の赤系アクセントを活かしつつ、背景はオフホワイト〜ライトグレー、信頼パートは濃紺〜チャコールで締める
- ガラス感やぼかしは控えめにし、“質感”は写真とタイポグラフィで作る

## 特に改善したいポイント

### 1. ヒーロー
- 汎用的な採用コピーをやめ、TCG業界・ショップ文脈が一読で伝わる見出しにする
- 例:
  - 「カードゲームで生きていきたい人へ」
  - 「好きなカードで、ちゃんと働く」
  - 「遊ぶ側から、支える側へ。」
- 右側は抽象モックや空白ではなく、店舗・イベント・作業風景などの実写を使う
- CTAは主従を明確にする
- ヒーローで“GOOD GAMEで働くと何が違うのか”が3秒で伝わるようにする

### 2. 数値訴求
- ただ大きい数字を置くのではなく、何の数字かが一目で分かるようにする
- 可能ならラベル、注釈、根拠スペースを用意する
- 数字の見た目は強く、説明は静かに
- 例:
  - [平均月収]
  - [イベント運営実績]
  - [取扱タイトル数]
  - [入社後独り立ち目安]

### 3. 仕事内容 / 募集職種
- SaaSカードではなく、店舗・運営・査定・発送・イベントという職務の違いが分かるUIにする
- カードの見た目に subtle な“ショップ文法”を入れる
  例:
  - 値札風ラベル
  - カテゴリタグ
  - 仕切り罫線
  - 在庫ラベル
  - チェック項目
- ただしコスプレ的な装飾は避ける
- “全部同じ白カード”に見えないよう、役割の違いを見た目でも出す

### 4. インタビュー / 先輩の声
- きれいすぎる演出より、リアルで信頼できる見せ方を優先
- 顔写真、役割、入社背景、1日の流れ、印象的な一言など、ドキュメンタリー感を出す
- 黒背景は使ってよいが、“プロダクトLP感”より“現場の空気”を優先する
- テキスト量が多い場合でも読みやすく整理する

### 5. FAQ / 募集要項
- 情報整理を最優先
- ここは演出より読みやすさ
- 採用不安の解消が目的なので、装飾は抑えめで良い
- スマホでも一気に読めるようにする

### 6. 最終CTA
- 現行のダークな締め方は活かしてよい
- ただし最後まで読んだ人が安心して応募できるように、応募前の小さな不安を解消する補助文を入れる
- 「応募する」だけでなく、「まず話を聞く」「見学・相談」などが必要なら主従をつけて整理する

## ビジュアルルール
- 角丸は全体で均一にしすぎない。主要カードと補助要素で差をつける
- シャドウは控えめ、境界は薄い線と面の差で作る
- タイポグラフィは見出しを少し強く、本文は読みやすさ優先
- “全部同じ箱”に見えないよう、セクションごとにリズムを作る
- 余白は広めでよいが、間延びさせない
- 写真は暖色寄り、実店舗の空気、手元、棚、卓、配送、接客が伝わるものを優先
- 実写が入る前提で、プレースホルダーも「何の写真か」が分かる指示にする

## 禁止事項
- 抽象的な3Dイラスト
- 意味のないグラデーション背景
- 汎用SaaSカードの量産
- 根拠のない実績数字や社員コメントの創作
- 情報量が多いのに見た目だけ軽いLP
- トレカ文化を“ゲームっぽい装飾”だけで表現すること
- ただモダンなだけで、誰のLPか分からない状態
